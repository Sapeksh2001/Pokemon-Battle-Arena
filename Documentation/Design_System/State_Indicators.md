# State Indicators

## 1. Visual Cues
- **Turn Priority**: The active player's Move Panel glows with a subtle white border.
- **Selection**: Targeted enemies pulse when a damaging move is clicked.

## 2. Audio Cues (Tone.js)
- **Selection Click**: Short 440Hz square wave beep.
- **Health Critical**: Pulsing 100Hz sine wave when HP falls below 20%.
- **Super Effective**: Harmonized major chord (C-E-G).
- **Not Very Effective**: Low-frequency dissonant noise burst.
