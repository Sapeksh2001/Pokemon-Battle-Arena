# Design Principles

## 1. Aesthetic: Modern Retro
The project balances nostalgia for Game Boy Era visuals with the performance and smoothness of modern web standards.

- **High Contrast**: Primary UI elements use deep blacks, vibrant reds/blues, and stark whites for maximum legibility.
- **Pixel Perfection**: Assets are scaled using `image-rendering: pixelated` to maintain crisp edges.
- **Zero Friction**: Avoid deep nesting or complex menus. All essential battle information should be visible in a single 16:9 view.

## 2. Emotional Response
- **Urgency**: Battle HP bars sweep smoothly, and log text types out to simulate a living narrator.
- **Impact**: Damage is reflected through screen shakes and sprite flashes.
