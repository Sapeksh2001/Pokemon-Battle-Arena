# Icon System

## 1. Type Icons
- **Mapping**: Each Pokémon Type (Fire, Water, etc.) is mapped to a specific hexadecimal color and a minimalist SVG badge.
- **Scaling**: Icons are kept at a 1:1 aspect ratio inside Move Buttons.

## 2. Status Badges
- **BRN**: Red background, White text.
- **PSN**: Purple background, White text.
- **PAR**: Yellow background, Black text.
- **FRZ**: Cyan background, Black text.
- **SLP**: Grey background, White text.

## 3. UI Helpers
Using `lucide-react` for system icons:
- `Sword`: Multiplier/Attack indicator.
- `Shield`: Defense indicator.
- `Zap`: Speed indicator.
