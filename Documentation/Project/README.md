# Pokémon Battle Arena

## Introduction
Welcome to the Pokémon Battle Arena documentation. This project is a React-migrated, high-performance Pokémon battle simulator designed for instant multiplayer play.

## Quick Links

- **[Tech Stack](./TECH_STACK.md)**: The underlying architecture and dependencies.
- **[Technical Docs](../Technical/Architecture_Overview.md)**: Deep dive into the internal engine.

## Getting Started
1. **Host a Room**: Enter your name and click "Create Room".
2. **Join a Room**: Enter the 6-digit code shared by your opponent.
3. **Battle**: Choose your team and click "Start Game" when both are ready.
